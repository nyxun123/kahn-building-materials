## 🛠️ 可用工具

### 核心工具

#### `read_file`
读取文件内容
```json
{
  "name": "read_file",
  "arguments": {
    "path": "/Users/nll/Documents/可以用的网站/package.json"
  }
}
```

#### `write_file`
写入文件内容
```json
{
  "name": "write_file",
  "arguments": {
    "path": "/Users/nll/Documents/可以用的网站/README.md",
    "content": "# 项目说明\n\n这是一个墙纸胶企业官网项目。",
    "createDirectories": true,
    "overwrite": true
  }
}
```

#### `list_directory`
列出目录内容
```json
{
  "name": "list_directory",
  "arguments": {
    "path": "/Users/nll/Documents/可以用的网站/src",
    "showHidden": false,
    "sortBy": "name",
    "sortOrder": "asc"
  }
}
```

### 增强工具

#### `copy_file`
复制文件或目录
```json
{
  "name": "copy_file",
  "arguments": {
    "source": "/Users/nll/Documents/可以用的网站/src/components",
    "destination": "/Users/nll/Documents/可以用的网站/backup/components",
    "recursive": true,
    "collisionStrategy": "skip",
    "preserveTimestamps": true
  }
}
```

#### `delete_file`
删除文件或目录
```json
{
  "name": "delete_file",
  "arguments": {
    "path": "/Users/nll/Documents/可以用的网站/temp",
    "recursive": true,
    "confirmationStrategy": "safe_mode",
    "createBackup": true,
    "dryRun": false
  }
}
```
