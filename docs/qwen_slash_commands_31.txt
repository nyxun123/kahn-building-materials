# Qwen 31个斜杠命令说明

根据提供的截图，以下是Qwen的31个斜杠命令：

1. `/about` - 显示版本信息
2. `/agents` - 管理用于专门任务委派的子代理
3. `/approval-mode` - 查看或更改工具使用的批准模式
4. `/auth` - 更改认证方法
5. `/bug` - 提交错误报告
6. `/chat` - 管理对话历史
7. `/clear` - 清除屏幕和对话历史
8. `/compress` - 通过替换为摘要来压缩上下文
9. `/config` - 配置设置
10. `/context` - 管理上下文
11. `/debug` - 调试模式
12. `/docs` - 显示文档
13. `/edit` - 编辑文件
14. `/exit` - 退出
15. `/export` - 导出当前会话
16. `/find` - 在代码库中查找
17. `/git` - Git操作
18. `/help` - 显示帮助信息
19. `/import` - 导入会话
20. `/list` - 列出可用工具
21. `/load` - 加载已保存的会话
22. `/model` - 管理模型
23. `/new` - 开始新会话
24. `/prompt` - 设置提示
25. `/quit` - 退出
26. `/read` - 读取文件
27. `/redo` - 重做上一个操作
28. `/save` - 保存当前会话
29. `/search` - 搜索文件内容
30. `/settings` - 显示设置
31. `/undo` - 撤销上一个操作

## 使用说明
- 所有命令都以`/`开头
- 命令后可跟参数，用空格分隔
- 这些命令主要用于AI开发助手的交互和管理功能
- 部分命令如`/edit`, `/read`, `/search`等用于文件操作
- `/(1/31)`表示这是第1个命令，共31个命令
