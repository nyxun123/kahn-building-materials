# Blog Image Credits

This file is the source-of-truth for external blog image attribution.

## Rules
- Prefer local real photos under `public/images` and `public/images/blog`.
- If using external images, only use copyright-clear sources (for example Unsplash/Pexels).
- Every external image URL used in blog content must be listed here before publishing.
- Do not use obvious AI-style visuals or suspicious AI-named assets (`generated`, `ai`, `midjourney`, `dalle`, `stable-diffusion`).
