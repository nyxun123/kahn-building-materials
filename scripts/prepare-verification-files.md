# 验证文件准备指南

当您在搜索引擎 Webmaster 工具中验证网站所有权时，系统会要求您：

## 方法 1：HTML 文件验证（推荐）

搜索引擎会提供一个 HTML 文件（如 `google1234567890.html`），您需要：
1. 将文件保存到 `public/` 目录
2. 告诉我文件名，我会帮您添加到代码并部署

## 方法 2：HTML 标签验证

搜索引擎会提供一个 HTML 标签（如 `<meta name="google-site-verification" content="xxx" />`），您需要：
1. 复制完整的标签
2. 告诉我标签内容，我会帮您添加到 `index.html` 并部署

## 方法 3：DNS 验证

搜索引擎会要求您在 DNS 中添加 TXT 记录，这需要在您的域名管理后台操作。

---

**建议流程**：
1. 您先访问搜索引擎的 Webmaster 工具
2. 选择验证方法（推荐 HTML 文件或 HTML 标签）
3. 将文件或标签内容告诉我
4. 我帮您添加到代码并部署
5. 您回到 Webmaster 工具点击"验证"
