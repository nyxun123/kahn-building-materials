## 图片规格建议

### 产品图片
- 建议尺寸：800x600 像素
- 格式：JPG 或 PNG
- 文件大小：不超过 500KB

### 图标类图片
- 建议尺寸：200x200 像素
- 格式：PNG（支持透明背景）
- 文件大小：不超过 100KB

### 横幅大图
- 建议尺寸：1200x600 像素
- 格式：JPG
- 文件大小：不超过 1MB

## 使用说明

1. 将所有实际图片文件重命名为上述列表中的对应文件名
2. 确保图片格式正确（JPG/PNG）
3. 优化图片大小以提高网站加载速度
4. 如需添加新图片，请同步更新HTML文件中的图片引用

## 占位图使用

在真实图片准备好之前，网站会使用CSS生成的占位图显示。这些占位图会在真实图片加载完成后自动替换。

## 注意事项

- 所有图片文件名请使用英文小写字母和连字符
- 避免使用中文或特殊字符作为文件名
- 建议对图片进行压缩优化以提升网站性能
- 如需修改图片尺寸，请保持宽高比例一致
