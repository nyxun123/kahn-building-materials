## 🎯 可能的问题

### 问题1: Token签名验证失败

**原因**: JWT_SECRET不一致
- 登录时使用的secret和验证时使用的secret不一致
- 导致签名验证失败

### 问题2: Token过期时间计算错误

**原因**: 时间同步问题
- 服务器时间和客户端时间不同步
- 导致token被认为已过期

### 问题3: Token类型错误

**原因**: Token类型不是'access'
- 登录时保存的不是access token
- 或者token的type字段不正确
