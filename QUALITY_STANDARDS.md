# Agents团队功能设计规范

## 概述
为杭州卡恩新型建材有限公司网站项目设计一套完整的Agents团队架构，实现模块化开发和维护。

## 核心代理 (Main Agents)

### 1. 前端开发代理 (Frontend Development Agent)
**职责**:
- React组件开发和维护
- UI/UX实现和优化
- 响应式设计适配
- 多语言界面实现

**技能要求**:
- React/Vite/TypeScript
- Tailwind CSS/Radix UI
- react-i18next
- 组件库开发

### 2. 后端API代理 (Backend API Agent)
**职责**:
- Cloudflare Workers API开发
- RESTful接口设计和实现
- 数据验证和错误处理
- 性能优化

**技能要求**:
- JavaScript/TypeScript
- Cloudflare Workers
- RESTful API设计
- 数据库集成

### 3. 数据库代理 (Database Agent)
**职责**:
- D1数据库设计和维护
- 数据模型优化
- 查询性能调优
- 数据迁移管理

**技能要求**:
- SQL数据库设计
- Cloudflare D1
- 数据建模
- 性能优化

### 4. 运维代理 (DevOps Agent)
**职责**:
- 部署流程管理
- 环境配置和维护
- 监控和告警设置
- 性能分析

**技能要求**:
- Cloudflare部署
- CI/CD流程
- 监控工具使用
- 性能调优

### 5. 安全代理 (Security Agent)
**职责**:
- 系统安全防护
- 认证授权机制
- 数据安全保护
- 安全漏洞检测

**技能要求**:
- Web安全最佳实践
- 认证授权机制
- 数据加密
- 安全审计

### 6. 测试代理 (Testing Agent)
**职责**:
- 测试用例设计和执行
- 自动化测试实施
- 性能测试执行
- 质量保证

**技能要求**:
- 测试框架使用
- 自动化测试
- 性能测试工具
- 质量评估

2. 定期进度同步
